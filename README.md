# LRC Time Shifter

A compact and powerful Python tool for playing music, as well as creating, editing, and synchronizing lyrics (LRC)

## Features
* **Time Shifting:** Easily correct lyric delays—globally or just for specific lines
* **Sync from Scratch:** An immersive mode that allows you to manually stamp timestamps on the fly onto raw text while listening to the track
* **Metadata Integration:** Read and save synchronized lyrics directly inside your audio files
* **Tag and Cover Editor:** A convenient window for editing audio metadata (Artist, Title, Album, etc.) and embedding cover art with a live preview (it can also extract cover picture and save it as imagine)
* **Smooth Navigation and Design:** A precise progress bar (powered by VLC), extensive hotkeys, and full appearance customization (many themes with the option to tweak every single color)
